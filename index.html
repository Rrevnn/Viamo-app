💰 Стипендии

Реальные возможности учиться бесплатно или получать деньги за учёбу

🇮🇹
DSU — Итальянская стипендия по нуждаемости
До €9,000/год + бесплатное общежитие
Доступно иностранцам
Италия платит иностранным студентам если доход семьи ниже определённого порога. Студенты с низким доходом платят €0 за обучение и получают деньги на жизнь.
📋 Шаги: 1) Поступи в вуз → 2) ISEE Parificato в консульстве → 3) Подай на DSU → 4) Получи решение
🇭🇺
Stipendium Hungaricum
Полное покрытие: обучение + €500/мес + общежитие
Дедлайн: январь
📋 Шаги: 1) stipendiumhungaricum.hu → 2) Выбери вузы → 3) Загрузи документы до января → 4) Собеседование
🇩🇪
DAAD — Немецкая служба академических обменов
€850–1,200/мес + обучение бесплатно
Конкурс высокий
📋 Шаги: 1) daad.de → 2) Поступи в немецкий вуз → 3) Подай на DAAD параллельно
🇪🇺
Erasmus+ Mundus
€1,000–1,400/мес + полное покрытие
Для магистратуры
📋 Шаги: 1) eacea.ec.europa.eu → 2) Подай единую заявку → 3) Нужен сильный профиль
🇹🇷
Türkiye Scholarships
Полное покрытие + карманные деньги
Дедлайн: февраль
📋 Шаги: 1) turkiyeburslari.gov.tr → 2) Зарегистрируйся → 3) Онлайн-собеседование → 4) Результат в июне
🇰🇷
GKS — Korean Government Scholarship
€800/мес + обучение + жильё + курс корейского
Дедлайн: март
📋 Шаги: 1) studyinkorea.go.kr → 2) Выбери вузы → 3) Подай через посольство Кореи
🇨🇳
CSC — Chinese Government Scholarship
Полное покрытие + €500/мес
Дедлайн: март
📋 Шаги: 1) campuschina.org → 2) Выбери вуз → 3) Подай через посольство Китая
